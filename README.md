# Easy-Blog  
一个基于ssm的博客系统，使用到相关技术有spring,mybatis,shiro,ajax,freemarker,wangEditor等。  
运行：  
1.mysql创建数据库easy-blog，，使用easy-blog.sql创建表。  
2.将easy-blog.war放置在tomcat7下的webapps。  
3.启动tomcat即可访问，首页地址localhost:8080/easy-blog。
